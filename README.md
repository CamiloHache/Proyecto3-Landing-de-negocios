# Proyecto Memorice

Landing page desarrollada para el Módulo 3 del curso de Desarrollo Web - UDD.

## Descripción del Proyecto

Proyecto Memorice es una landing page para una iniciativa de diseño que se dedica a rescatar y difundir la gráfica chilena relacionada con la memoria reciente de la política chilena. El sitio presenta información sobre el proyecto, sus valores y un catálogo de productos que incluye pines y bolsas con diseños históricos.

El proyecto surgió a partir de la conmemoración de los 50 años del golpe cívico-militar, con el objetivo de mantener vigente la memoria visual de la política chilena.

## Tecnologías Utilizadas

- **HTML5**: Para la estructura del sitio
- **CSS3**: Para los estilos (combiné CSS personalizado en `main.css` con TailwindCSS vía CDN)
- **JavaScript**: Para funcionalidades interactivas como el menú móvil y validación de formularios

## Secciones del Sitio

### Header
Incluye el logo del proyecto y el menú de navegación principal con enlaces a las secciones: Inicio, Nosotros, Productos y Contacto. El menú es responsive y tiene una versión móvil con botón hamburguesa.

### Main (Hero)
Sección principal de presentación con título, descripción del proyecto y un botón call-to-action. Tiene una imagen de fondo con overlay para que el texto sea legible.

### Products
Muestra el catálogo de productos en un grid responsivo. Actualmente incluye pines y bolsas con sus imágenes, categorías y descripciones.

### Footer
Información de derechos reservados y enlaces a redes sociales (Instagram y Facebook).

## Prototipo

El prototipo inicial fue diseñado usando Draw.io y se encuentra en `assets/img/prototipo-proyecto-memorice.drawio`.

## Desarrollo del Proyecto

Las primeras versiones del sitio fueron desarrolladas por mí desde cero, usando HTML5 y CSS3 principalmente. Luego, trabajé con la diseñadora del proyecto y los dueños para hacer ajustes y correcciones según sus feedback, especialmente en cuanto a colores, tipografía y algunos elementos visuales como las texturas de fondo.

## Responsive Design

El sitio está adaptado para diferentes tamaños de pantalla usando media queries. En móviles el menú se convierte en un menú hamburguesa y los grids se adaptan a una sola columna.
